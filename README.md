# Validação de Senha em Java

Este projeto consiste em um programa em Java que realiza a validação de uma senha digitada pelo usuário.

O programa lê repetidamente valores inteiros até que a senha correta seja informada.  
Enquanto a senha estiver incorreta, a mensagem **"Senha Invalida"** é exibida.  
Quando a senha correta (**2002**) é digitada, o programa exibe **"Acesso Permitido"** e encerra a execução.

## Tecnologias utilizadas
- Java
- Scanner
- Estruturas de repetição (`while`)
- Estruturas condicionais (`if/else`)

## Objetivo
Exercitar lógica de programação, leitura de dados e controle de fluxo em Java.
